<code>

Result code Element

Attribute

NameDescriptionOmission
---

Value

Description
Result code
  • 0 … Success
  • 1100 … Printer off line(On error)
  • 2001 … Printer busy(In use with another device: Need to resend data)
Example
<root>
　　<success>true</success>
　　<code>0</code>
　　<status>2386000000000000000000</status>
